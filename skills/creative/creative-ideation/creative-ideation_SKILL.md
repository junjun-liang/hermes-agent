# 创意构想

通过创意约束生成项目想法。约束 + 方向 = 创造力。

## 工作原理

1. **从下方库中选择一个约束** — 随机的，或匹配用户的领域/心情
2. **广义解释** — 编码提示可以变成硬件项目，艺术提示可以变成CLI工具
3. **生成3个满足约束的具体项目想法**
4. **如果他们选择一个，就构建它** — 创建项目、编写代码、发布它

## 规则

每个提示都尽可能广义地解释。"这包括X吗？" → 是的。提示提供方向和轻度约束。没有这两者，就没有创造力。

## 约束库

### 面向开发者

**解决自己的痛点：**
构建您本周希望存在的工具。少于50行。今天发布。

**自动化烦人的事情：**
您工作流中最繁琐的部分是什么？用脚本消除它。花两小时修复一个每天浪费您五分钟的问题。

**应该存在的CLI工具：**
想想您希望可以输入的命令。`git undo-that-thing-i-just-did`。`docker why-is-this-broken`。`npm explain-yourself`。现在构建它。

**没有新东西，只有胶水：**
完全用现有API、库和数据集制作东西。唯一的原创贡献是您如何连接它们。

**弗兰肯斯坦周：**
把做X的东西变成做Y。播放音乐的git仓库。生成诗歌的Dockerfile。发送赞美的cron作业。

**减法：**
在代码库崩溃前可以移除多少？将工具剥离到最小可用功能。删除直到只剩本质。

**高概念，低努力：**
深刻的想法，轻松执行。概念应该出色。实现应该花一个下午。如果花更长时间，您想太多了。

### 面向制作者和艺术家

**公然复制某物：**
选择您欣赏的东西 — 工具、艺术品、界面。从零重新创建它。学习在于您的版本和他们的版本之间的差距。

**一百万个某物：**
一百万既多又不那么多。一百万像素是一张1MB照片。一百万次API调用是周二。任何事物在规模上都变得有趣。

**制作会消亡的东西：**
每天失去一个功能的网站。忘记的聊天机器人。倒数到虚无。关于腐烂、消亡或放手的练习。

**做大量数学：**
生成几何、着色器高尔夫、数学艺术、计算折纸。是时候重新学习arcsin了。

### 面向任何人

**文本是通用接口：**
构建只有文本作为界面的东西。没有按钮、没有图形，只有输入和输出的文字。文本几乎可以进出任何东西。

**从妙语开始：**
想想会是有趣句子的东西。倒推让它实现。"我教我的恒温器煤气灯效应我" → 现在构建它。

**敌对UI：**
有意制作难以使用的东西。需要47个条件的密码字段。每个标签都在撒谎的表单。评判您命令的CLI。

**重来一次：**
记住旧项目。从零再做一次。不要看原版。看看您思维的变化。

查看 `references/full-prompt-library.md` 获取30+额外约束，涵盖通信、规模、哲学、转换等。

## 将约束匹配给用户

| 用户说 | 选择自 |
|-----------|-----------|
| "我想构建点东西"（无方向） | 随机 — 任何约束 |
| "我正在学习[语言]" | 公然复制某物，自动化烦人的事情 |
| "我想要奇怪的东西" | 敌对UI，弗兰肯斯坦周，从妙语开始 |
| "我想要有用的东西" | 解决自己的痛点，应该存在的CLI，自动化烦人的事情 |
| "我想要美丽的东西" | 做大量数学，一百万个某物 |
| "我倦怠了" | 高概念低努力，制作会消亡的东西 |
| "周末项目" | 没有新东西只有胶水，从妙语开始 |
| "我想要挑战" | 一百万个某物，减法，重来一次 |

## 输出格式

```
## 约束：[名称]
> [约束，一句话]

### 想法

1. **[一行推介]**
   [2-3句：您将构建什么以及为什么有趣]
   ⏱ [周末/周/月] • 🔧 [技术栈]

2. **[一行推介]**
   [2-3句]
   ⏱ ... • 🔧 ...

3. **[一行推介]**
   [2-3句]
   ⏱ ... • 🔧 ...
```

## 示例

```
## 约束：应该存在的CLI工具
> 想想您希望可以输入的命令。现在构建它。

### 想法

1. **`git whatsup` — 显示您不在时发生了什么**
   比较您最后的活跃提交到HEAD并总结更改内容、
   谁提交了以及合并了什么PR。就像仓库的晨会。
   ⏱ 周末 • 🔧 Python, GitPython, click

2. **`explain 503` — 给人类看的HTTP状态码**
   管道任何状态码或错误消息并获取纯英文解释
   带常见原因和修复方法。从策划的数据库拉取，不是LLM。
   ⏱ 周末 • 🔧 Rust或Go，静态数据集

3. **`deps why <package>` — 为什么这在我的依赖树中**
   追踪传递依赖回到拉取它的直接依赖。
   一条命令回答"为什么我有47份lodash"。
   ⏱ 周末 • 🔧 Node.js, npm/yarn lockfile解析
```

用户选择一个后，开始构建 — 创建项目、编写代码、迭代。
