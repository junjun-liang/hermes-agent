## 1. 使用 curl 命令

### 1.1 基本聊天请求

```bash
# 发送聊天消息
curl -X POST http://localhost:8001/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "message": "你好，请介绍一下你自己"
  }' | python -m json.tool
```

**响应示例：**
```json
{
    "session_id": "abc123",
    "response": "你好！我是 Hermes-Agent AI 助手...",
    "model": "dashscope/deepseek-v4-flash",
    "duration": 2.5,
    "cost_usd": 0.001
}
```

### 1.2 继续对话

```bash
# 使用之前的 session_id 继续对话
curl -X POST http://localhost:8001/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "message": "继续刚才的话题",
    "session_id": "abc123"
  }' | python -m json.tool
```

### 1.3 带参数的聊天

```bash
# 指定模型和最大迭代次数
curl -X POST http://localhost:8001/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "message": "帮我创建一个 Python 快速排序算法",
    "model": "dashscope/deepseek-v4-flash",
    "max_iterations": 50,
    "toolsets": ["file", "code_execution"]
  }' | python -m json.tool
```

## 2. 使用 Python CLI 脚本

项目提供了一个完整的命令行聊天脚本 `cli_chat.py`。

### 2.1 使用方法

```bash

python fastapi_server/cli_chat.py

# 发送单条消息

# 继续之前的对话
python fastapi_server/cli_chat.py "继续" abc123

# 显示详细信息（模型、耗时、成本）
```

## 3. 使用交互式聊天

创建一个简单的交互式聊天脚本：

```python
#!/usr/bin/env python3
"""简单的交互式聊天脚本"""
import requests

BASE_URL = "http://localhost:8001/api/v1"

print("🤖 Hermes-Agent 聊天（输入 quit 退出）")
session_id = None

while True:
    user_input = input("\n👤 你：").strip()
    if user_input.lower() in ['quit', 'exit', 'q']:
        print("👋 再见！")
        break
    
    response = requests.post(
        f"{BASE_URL}/chat/completions",
        json={"message": user_input, "session_id": session_id}
    )
    
    result = response.json()
    session_id = result.get('session_id')
    print(f"🤖 AI: {result.get('response')}")
```

## 6. 常见问题

### 6.1 401 未授权错误

**错误：**
```json
{
    "error": "未提供认证凭据"
}
```

**解决方法：**

1. **开发环境**：确保 `.env` 文件中 `API_KEYS=` 为空
2. **生产环境**：在请求头中添加 API Key
   ```bash
   curl -H "X-API-Key: your-api-key-here" ...
   ```

### 6.2 404 未找到错误

**错误：**
```json
{
    "error": "404 Not Found"
}
```

**原因：** 端点路径错误

**解决：** 使用正确的端点 `/api/v1/chat/completions`（不是 `/api/v1/chat`）

### 6.3 500 服务器内部错误

**可能原因：**
- API Key 无效或过期
- LLM 服务不可用
- 数据库连接问题

**解决方法：**
1. 检查服务器日志查看详细错误
2. 验证 API Key 是否有效
3. 检查服务是否正常运行

### 6.4 连接错误

**错误：**
```
Connection refused
```

**解决方法：**
```bash
# 检查服务是否运行
curl http://localhost:8001/api/v1/health

# 如果服务未运行，启动服务
cd /home/meizu/Documents/my_agent_project/hermes-agent
uvicorn fastapi_server.main:app --host 0.0.0.0 --port 8001
```

## 7. 最佳实践

### 7.1 会话管理

- **保存 session_id**：每次请求后保存返回的 `session_id`，用于继续对话
- **定期清理**：删除不需要的会话，避免数据库过大
- **会话标题**：为重要会话设置有意义的标题

### 7.2 错误处理

在脚本中添加错误处理：

```python
try:
    response = requests.post(url, json=data, timeout=120)
    response.raise_for_status()
    result = response.json()
except requests.exceptions.ConnectionError:
    print("无法连接到服务")
except requests.exceptions.Timeout:
    print("请求超时")
except Exception as e:
    print(f"错误：{e}")
```

### 7.3 性能优化

- **合理设置 max_iterations**：根据任务复杂度设置，避免不必要的迭代
- **使用会话延续**：避免重复发送上下文信息
- **批量处理**：对于多个简单问题，考虑使用批量 API（如果支持）

## 8. 完整示例

### 8.1 Bash 脚本示例

```bash
#!/bin/bash
# chat.sh - 简单的聊天脚本

MESSAGE="$1"
SESSION_ID="$2"
BASE_URL="http://localhost:8001/api/v1"

if [ -z "$MESSAGE" ]; then
    echo "用法：$0 \"消息内容\" [session_id]"
    exit 1
fi

# 构建 JSON
if [ -n "$SESSION_ID" ]; then
    JSON="{\"message\": \"$MESSAGE\", \"session_id\": \"$SESSION_ID\"}"
else
    JSON="{\"message\": \"$MESSAGE\"}"
fi

# 发送请求
RESPONSE=$(curl -s -X POST "$BASE_URL/chat/completions" \
    -H "Content-Type: application/json" \
    -d "$JSON")

# 解析响应
AI_RESPONSE=$(echo "$RESPONSE" | python -c "import sys, json; print(json.load(sys.stdin).get('response', 'No response'))")
NEW_SESSION_ID=$(echo "$RESPONSE" | python -c "import sys, json; print(json.load(sys.stdin).get('session_id', ''))")

echo "🤖 AI: $AI_RESPONSE"
echo "💡 Session ID: $NEW_SESSION_ID"
```

**使用方法：**
```bash
chmod +x chat.sh
./chat.sh "你好"
./chat.sh "继续" "abc123"
```

### 8.2 Python 批量聊天示例

```python
#!/usr/bin/env python3
"""批量聊天示例"""
import requests

BASE_URL = "http://localhost:8001/api/v1"
session_id = None

questions = [
    "什么是快速排序？",
    "请用 Python 实现快速排序",
    "解释一下时间复杂度",
]

for question in questions:
    print(f"\n👤 Q: {question}")
    
    response = requests.post(
        f"{BASE_URL}/chat/completions",
        json={"message": question, "session_id": session_id}
    )
    
    result = response.json()
    session_id = result.get('session_id')
    print(f"🤖 A: {result.get('response')}")

print(f"\n💡 会话 ID: {session_id}")
```
